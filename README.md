Pulseapp Messaging
A clean, modern WhatsApp-style messaging web app built with HTML, CSS, and JavaScript.

✨ Features
Add contacts with mobile number
Message reactions (15+ emojis)
Message status (sent → delivered → read)
Typing indicator
Star and delete messages
Block contacts
Dark mode toggle
Last seen / online status
Attachment support
🚀 Live Demo
[Coming soon]

💻 How to Run Locally
Clone or download this repository
Open index.html in any modern browser
No installation or build step required
🛠 Tech Stack
HTML5
CSS3
Vanilla JavaScript
Fully static (no backend needed)
